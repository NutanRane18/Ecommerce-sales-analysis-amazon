# 🛍️ E-Commerce Sales Intelligence | Amazon Dataset

## 🎯 Objective

To explore and analyze Amazon sales data and transform raw transactional data into actionable business insights using data analytics and visualization tools.

---

## 📦 Dataset Summary

This project uses an Amazon sales dataset containing information on:

* Orders and product categories
* Customer regions
* Pricing, discounts, and revenue
* Payment methods and customer ratings

---

## 🔍 Problem Statement

E-commerce businesses generate large volumes of data but often struggle to:

* Understand product performance across categories
* Measure the real impact of discounts on revenue
* Track customer purchasing trends across regions
* Identify key drivers of revenue growth

This project addresses these challenges through structured analysis and dashboards.

---

## ⚙️ Approach

### 1️⃣ Data Preparation

* Cleaned and transformed raw data using SQL
* Standardized date formats and extracted monthly trends
* Created calculated fields like **gross revenue, net revenue, and discount value** 

### 2️⃣ KPI Development

* Total Revenue, Total Orders, Quantity Sold
* Average Order Value (AOV)
* Customer Ratings

### 3️⃣ Data Visualization

* Built dashboards using Power BI, Tableau, and Excel
* Applied filters for region, category, and time-based analysis
* Enabled interactive exploration of sales performance

---

## 📊 Key Findings

### 📈 Sales Trends

* Revenue remained stable between **$2.5M – $2.9M monthly**
* No extreme seasonal spikes, indicating steady demand

### 🛒 Category Insights

* **Beauty and Books** emerged as top-performing categories
* Sales distribution is relatively balanced across categories

### 🌍 Regional Insights

* **Asia recorded the highest contribution (~25%)**
* Other regions showed consistent performance

### 💳 Payment Behavior

* Digital payments dominate, with **Wallet slightly leading (~20%)**
* Balanced usage across multiple payment methods

### 🎯 Discount Analysis

* Increasing discounts does not guarantee higher revenue
* Moderate discounts showed better performance compared to high discounts
